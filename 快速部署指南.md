# 快速部署到腾讯云 COS

## 方法一: 手动部署 (推荐,5分钟搞定)

### 1. 创建存储桶
访问: https://console.cloud.tencent.com/cos5
- 点击"创建存储桶"
- 名称: `metabolism-analysis-20260319`
- 地域: 广州或上海
- 访问权限: **公共读**
- 点击确定

### 2. 上传文件
- 进入存储桶
- 点击"上传文件"
- 选择项目中的 `index.html`
- 点击上传

### 3. 开启静态网站
- 存储桶设置 → 静态网站
- 开启静态网站功能
- 索引文档: `index.html`
- 保存

### 4. 访问链接
```
https://metabolism-analysis-20260319.cos.ap-guangzhou.myqcloud.com/index.html
```

---

## 方法二: 使用脚本部署

### 1. 获取腾讯云密钥
- 访问: https://console.cloud.tencent.com/cam/capi
- 获取 `SecretId` 和 `SecretKey`
- 创建存储桶(同方法一第一步)

### 2. 设置环境变量
在 PowerShell 中执行:
```powershell
$env:TENCENT_SECRET_ID="你的SecretId"
$env:TENCENT_SECRET_KEY="你的SecretKey"
```

### 3. 运行部署脚本
```powershell
node deploy.js
```

---

## 密钥权限要求
密钥需要具备以下权限:
- `PutObject` (上传文件)
- `PutObjectACL` (设置文件权限)

---

## 常见问题

**Q: 上传后无法访问?**
A: 检查存储桶权限是否设置为"公共读"

**Q: 如何更新文件?**
A: 直接重新上传同名文件覆盖即可

**Q: 能否使用自定义域名?**
A: 可以,在存储桶域名管理中绑定
