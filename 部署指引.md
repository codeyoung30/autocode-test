# Tencent Cloud Toolkit 部署指引

## 部署目标
将 `metabolism_analysis_offline.html` 部署到公网进行测试

## 操作步骤

### 方法一: 使用 VS Code 中的 Tencent Cloud Toolkit

1. **打开 VS Code 扩展面板**
   - 点击左侧扩展图标或按 `Ctrl+Shift+X`
   - 搜索 "Tencent Cloud Toolkit"
   - 确认扩展已安装并启用

2. **登录腾讯云**
   - 点击左侧腾讯云图标(云朵形状)
   - 点击 "Login" 进行授权登录
   - 使用腾讯云账号扫码或密码登录

3. **部署静态网站**
   - 在 VS Code 资源管理器中右键点击 `metabolism_analysis_offline.html`
   - 选择 "Tencent Cloud Toolkit: Deploy to Static Website"
   - 或者在命令面板 (`Ctrl+Shift+P`) 中搜索该命令

4. **配置部署选项**
   - 选择云对象存储 (COS) 或静态网站托管
   - 选择或创建存储桶
   - 设置访问路径(建议直接使用文件名)
   - 确认部署

5. **获取访问链接**
   - 部署成功后会显示公网访问链接
   - 复制链接即可分享给他人测试

### 方法二: 使用腾讯云控制台手动部署

1. 登录腾讯云控制台: https://console.cloud.tencent.com/

2. 进入对象存储 COS 服务

3. 创建存储桶:
   - 选择公开读权限
   - 选择就近地域(如广州、上海)
   - 设置访问域名(静态网站托管)

4. 上传文件:
   - 点击"上传文件"
   - 选择 `metabolism_analysis_offline.html`
   - 等待上传完成

5. 配置静态网站:
   - 在存储桶设置中开启静态网站
   - 设置首页文件为 `metabolism_analysis_offline.html`

6. 获取访问链接:
   - 在域名管理中查看公网访问地址
   - 格式通常为: `https://<bucket-name>.cos.<region>.myqcloud.com/metabolism_analysis_offline.html`

## 文件说明

- **metabolism_analysis_offline.html**: 离线版本,包含所有Chart.js依赖,无需外网加载
- 文件大小: 约 60KB
- 支持浏览器: Chrome, Edge, Firefox, Safari 等现代浏览器

## 测试建议

1. 在手机浏览器中打开链接测试响应式布局
2. 在不同浏览器中测试兼容性
3. 检查图表交互功能是否正常
4. 验证所有计算功能准确性

## 常见问题

**Q: 部署后无法访问?**
A: 检查存储桶权限是否设置为"公共读"

**Q: 如何更新文件?**
A: 直接重新上传同名文件覆盖即可

**Q: 能否使用自定义域名?**
A: 可以,在COS域名管理中绑定自定义域名并配置CNAME
